# proxy-tester

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/pdsdevelopment/proxy-tester.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run It `index.js`
   ```js
   node index.js [proxy-list-file] [output-file] [custom-check-host] [custom-timeout]
